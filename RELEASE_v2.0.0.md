Prospectors Plus v2.0.0 — the big rework

A top-to-bottom rework and rebrand. New look, invite-only access, near-zero setup,
and notifications you control. If you used the old version, everything you knew is
here — just faster to set up and nicer to use.

### ✨ New

- **Invite-only access codes.** The app opens to a splash, then an access-code
  screen. Enter the code you were given and that PC is remembered for good. Codes
  are managed centrally, so access can be added or revoked anytime.
- **One-click auto-calibrate.** Open Prospecting and hit **Auto-calibrate** — it
  finds the game window and places every spot for you. New users don't have to
  calibrate by hand at all.
- **Custom Discord notifications.** Toggle exactly which DMs you get — start, stop,
  periodic stats, safe-stop, recoveries, errors — each on its own switch.
- **Calibration export / import.** Save your calibration to a file and load it back
  after a reinstall or on another PC, instead of redoing it.
- **Manual calibration entry.** Type exact X / Y coordinates and colours for any
  spot, with a live colour swatch.

### ⚡ Improved

- **Brand-new interface** — a sleek, dark, professional look with a new diamond
  logo (it's panning, not mining!), smooth animations, and an animated splash.
- **Much more reliable Roblox detection** — finds the game window even when the
  title varies, and the macro re-aims itself to the live window on start, so
  calibration survives the window being moved or resized.
- **Better calibration** — capture a spot by clicking it *or* hovering and pressing
  Enter; Esc to cancel; clear messages when something goes wrong.
- **In-app updates** — the app tells you when a new version is out and links
  straight to the download. Your settings, builds, and calibration carry over.
- Every setting now has a plain-English tooltip.

### 🖥️ Install

1. Download **ProspectorsPlusSetup.exe** below.
2. Open it. If Windows shows a blue "Windows protected your PC" box, click
   **More info → Run anyway** (normal for new apps).
3. Launch from your desktop or Start menu, enter your access code, and you're in.

Windows 10 & 11 · free · macros are allowed in Prospecting — this tool only
automates clicks and key presses and never injects code or modifies the game.
