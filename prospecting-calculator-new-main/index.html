Manual (Sandshaking)
Use these values if you are shaking the pan yourself.
• Dig inputs are not used for manual shaking

Autopan / Macro
Use these values if a macro or autopan is shaking for you.
• n (digs to fill): how many digs are needed to fill a pan
• d (dig speed %): your dig speed from gear/upgrades

Important:
Luck and Capacity affect drop chance.
Shaking speed and dig speed affect time only.
Faster pans do not increase RNG - they just reach it sooner.
