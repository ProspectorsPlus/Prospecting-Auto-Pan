# Prospecting Calculator

A calculator and utility website for the Roblox game **Prospecting**.

## Features

- Mineral chance calculations
- Luck & capacity simulations
- Attempt probability calculations
- Time-to-find estimates
- 50% / 90% odds tracking
- Location filtering
- Rarity filtering
- Equipment/stat support
- Crafting information

## How It Works

The calculator uses the game's base mineral chances along with:

- Luck
- Pan capacity
- Effective rerolls
- Independent roll probability

to estimate:

- Chance per attempt
- Expected attempts
- Expected finds
- Time estimates
- Probability milestones

## Tech Stack

- HTML
- CSS
- JavaScript
