Prospectors Plus v1.0.0 — First public release

The first official release of Prospectors Plus: a smart auto-pan bot for
Prospecting that runs the whole farming loop for you — pan, dig, shake, deposit —
reading the screen so it keeps going on its own, even while you're away. Now a
one-click Windows installer, no Python or setup required.

### ✨ What's included

- **Full auto-pan loop** — pans, digs, shakes and deposits automatically, then
  repeats. You set it up once and let it run.
- **Reads the screen** — a supervisor watches key pixels to tell dirt, lava/water,
  the shake phase and an active dig apart, and does the right action for each.
- **Capacity-bar aware** — tracks your pan's fill bar so it digs while there's room
  and deposits when it's actually full, instead of looping blindly.
- **Self-recovery** — dig-probe landing, a momentum shake and layered break-out
  steps pull it out of bad spots so a session keeps going instead of stalling.
- **Relic timers** — set each relic's slot and cooldown (Solar Magnifier, Infernal
  Idol, etc.) and it re-activates them on schedule while it farms.
- **Discord DMs** — get a DM on start and stop, plus periodic session stats, so you
  can leave it running and still know how it's going. Just enter your username.
- **Live session stats** — cycles, pans-per-hour, recoveries and runtime, shown
  live in the app.
- **Auto-stop** — stop automatically after a set number of minutes or pans.
- **Tunable builds** — every timing is adjustable, and you can save full presets
  (e.g. a shards build) and switch between them in one click.
- **One-click calibration** — click the listed spots in-game once and the app
  learns where to look.
- **In-app updates** — checks for new versions on launch and shows a banner that
  links you straight to the download. Your settings and builds carry over.

### 🖥️ Install

1. Download **ProspectorsPlusSetup.exe** below.
2. Open it. If Windows shows a blue "Windows protected your PC" box, click
   **More info → Run anyway** (normal for new apps).
3. Launch from your desktop or Start menu.

### ✅ Good to know

Macros are allowed in Prospecting — this tool only automates clicks and key
presses and never injects code or modifies the game.
