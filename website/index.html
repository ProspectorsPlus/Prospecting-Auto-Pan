How to install

  1. Click Download for Windows above.
  2. Open the downloaded Prospectors Plus Setup.exe.
  3. If Windows shows a blue "Windows protected your PC" box, click More info → Run anyway (this is normal for new apps).
  4. Follow the installer, then launch from your desktop or Start menu.
