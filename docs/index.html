What it does

More than a click bot

It watches the game and reacts, so a session does not stall the moment something goes slightly wrong.

Reads the screen

A supervisor watches key pixels to tell dirt, lava, water, the shake phase and active digs apart, then picks the right action.

Capacity-bar aware

It tracks your pan fill bar so it knows when to keep digging and when it is actually time to deposit.

Recovers itself

Dig-probe landing, momentum shake, and layered break-out steps pull it out of bad spots so it keeps farming.

Relic timers

Set each relic slot and cooldown and it re-activates them on schedule: Solar Magnifier, Infernal Idol, or whatever you run.

Discord DMs

Get a DM on start and stop plus periodic session stats, so you can leave it running and still know how it is going.

Live stats and auto-stop

Cycles, pans per hour, recoveries and runtime update live. Set it to stop after a set number of minutes or pans.

Cycle page tuning

New in 4.0 — your whole pan loop drawn as a live diagram with a millisecond timeline graph. Drag a slider and the graph redraws; click any bar to jump to its exact setting.

Shards mode

New in 4.0 — exact-click digging with green dig-bar confirmation and an assume-full pipeline, built for one-click-fill shard farms.

Live HUD overlay

New in 4.0 — a draggable on-screen card showing the current stage (Digging, Hold S, Glide, Shaking), live stats and a find ticker while you farm.

Before you download

Requirements and notes

Nothing unusual. If you can run Prospecting you can run this.

Requirements

  • Windows 10/11 (64-bit) — or macOS on Apple Silicon or Intel
  • Roblox running in windowed or borderless mode
  • Mouse and keyboard control available (not remoted in)
  • Windows: no Python, nothing extra · macOS: Python 3 installed once

Good to know

  • Macros are allowed in Prospecting. This tool only automates clicks and key presses.
  • First run: Windows may show a SmartScreen warning (More info → Run anyway); macOS may say it "could not verify" the app (System Settings → Privacy & Security → Open Anyway). Normal for unsigned indie tools.
  • Access is invite-only: the app asks for a one-time access code on first launch, and each code locks to one computer.
  • Settings, calibration and saved builds are stored locally and survive updates.
  • The app controls your mouse and keyboard, so leave it running on its own while you are away.
Questions

FAQ

Are macros allowed in Prospecting?

Yes. Prospectors Plus only automates mouse clicks and key presses, the same inputs you would make yourself. It does not inject code, read game memory, or modify the game in any way.

Do I need Python or any setup?

On Windows, no — it is a normal installer: download, run, launch. On macOS, install Python 3 once (from python.org), then the app sets up the rest by itself on first launch.

Is there a Mac version?

Yes — new in 4.0. Download the .dmg and drag the app into Applications. The first time you open it, macOS may say it "could not verify" the app because it is a small unsigned indie tool — this is normal. Open System Settings → Privacy & Security, scroll down, and click Open Anyway. You only do this once. The finds and earnings dashboards are Mac-only for now.

Why does it ask for an access code?

Prospectors Plus is invite-only. Enter the code you were given and it unlocks on that computer — codes are machine-locked, so one code cannot be passed around between friends.

Will Windows warn me when I open it?

It may show a blue "Windows protected your PC" box because the app is new and unsigned. Click More info, then Run anyway once. This is normal for small indie tools and does not mean the app is harmful.

How do updates work?

The app checks for new versions on launch and shows a banner when one is available, linking you straight back here. Your settings, saved builds and calibration carry over automatically.

Can I use it while doing other things?

It controls the mouse and keyboard, so it works best on its own while you step away. Discord DMs let you check in without watching the screen.

Can I save my own build presets?

Yes. Every timing, relic slot and Discord setting can be saved as a named preset. The two included builds (Money and Money v2) are a starting point — save as many custom ones as you like and switch between them in one click.
