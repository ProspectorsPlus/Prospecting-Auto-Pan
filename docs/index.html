What it actually does

Point it at the game, calibrate a few spots once, and hit start. From then on it watches the pan and the capacity bar and drives the full farming cycle on its own.

Pan→ Dig→ Shake→ Deposit↺ repeats automatically

🧠 Reads the screen

A supervisor watches a handful of pixels to tell dirt, lava/water, the shake phase and an active dig apart, then does the right action for the state it's in.

📊 Capacity-bar aware

It tracks your pan's fill bar so it knows when to keep digging and when it's time to go deposit — instead of blindly looping.

🪄 Gets itself unstuck

Dig-probe landing, a momentum shake, and layered recovery/break-out steps pull it out of bad spots so a session doesn't just stall.

⏳ Relic timers

Set your relics (slot + cooldown) and it re-activates them on schedule while it farms — Solar Magnifier, Infernal Idol, whatever you run.

💬 Discord DMs

Get a DM when a session starts or stops, plus periodic stats — so you can leave it running and still know how it's going.

📈 Live session stats

Cycles completed, pans per hour, recoveries and runtime, shown live in the app while it works.

🛑 Auto-stop

Stop after a set number of minutes or pans, so it won't run forever when you forget about it.

🎛️ Tunable builds

Every timing is adjustable and you can save full presets (e.g. a shards build) and switch between them in one click.

Using it

  1. Open the app and calibrate once — click the listed spots in-game (pan, deposit, shake, dig trigger, capacity bar) so it knows where to look.
  2. Pick or tweak a build, set your relics, and (optional) type your Discord username for DMs.
  3. Press Start, then Ctrl+K in-game to begin. Hit Esc to stop anytime.
Macros are allowed in Prospecting. This tool only automates clicks and key presses — it does not inject code or modify the game.

Install

  1. Click Download for Windows at the top.
  2. Open ProspectorsPlusSetup.exe. If Windows shows a blue "Windows protected your PC" box, click More info → Run anyway (normal for new apps).
  3. Launch from your desktop or Start menu. Updates show up in-app automatically.
