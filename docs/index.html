What it does

More than a click bot

It watches the game and reacts — so a session doesn't stall the moment something goes slightly wrong.

Reads the screen

A supervisor watches key pixels to tell dirt, lava/water, the shake phase and an active dig apart, then does the right thing for each.

Capacity-bar aware

It tracks your pan's fill bar, so it knows when to keep digging and when it's actually time to deposit — instead of looping blindly.

Recovers itself

Dig-probe landing, a momentum shake and layered break-out steps pull it out of bad spots so it keeps farming instead of getting stuck.

Relic timers

Set each relic's slot and cooldown and it re-activates them on schedule while it works — Solar Magnifier, Infernal Idol, whatever you run.

Discord DMs

Get a DM on start and stop plus periodic stats, so you can leave it running and still know how the session is going.

Live stats & auto-stop

Cycles, pans-per-hour, recoveries and runtime update live — and you can have it stop after a set number of minutes or pans.

Good to know

FAQ

Are macros allowed in Prospecting?

Yes. Prospectors Plus only automates mouse clicks and key presses — the same inputs you'd make yourself. It does not inject code, read game memory, or modify the game in any way.

Do I need Python or any setup?

No. It's a normal Windows installer — download, run, and launch from your desktop or Start menu. Nothing else to install.

Will Windows warn me when I open it?

It may show a blue "Windows protected your PC" box because the app is new and unsigned. Click More info → Run anyway once. This is normal for small indie tools.

How do updates work?

The app checks for new versions on launch and shows a banner when one is available, linking you straight back here. Your settings, saved builds and calibration are kept between updates.

Does it work while I'm doing other things?

It controls the mouse and keyboard, so it runs best on its own while you're away. It sends Discord DMs so you can check in without watching it.
